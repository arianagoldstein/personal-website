# Personal Website

A personal website to replace an old Squarespace site. Built with modern web tooling.

## Goals
- Showcase personal projects with descriptions and links
- Highlight companies worked at
- Link to social profiles (GitHub, LinkedIn, etc.)

## Claude Code Usage
This project intentionally uses Claude Code skills and agents to demonstrate their value:
- `/simplify` — run after implementing a feature to review for quality and reuse
- `Explore` agent — use for codebase exploration before making changes
- `Plan` agent — use before implementing non-trivial features

## Development
- Run `npm run dev` to start the dev server
- Run `npm run lint` to lint
- Run `npm run build` to build for production
